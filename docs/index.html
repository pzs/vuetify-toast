Vuetify Toast Demo

Show 'hello' Show 'error' Show toast for 10 sec 

          vuetifyToast.success('Hello');

          vuetifyToast.error('Error');

          vuetifyToast.show({
            text: 'This is an information.',
            color: 'info',
            timeout: 10000,
            icon: 'info'
          });


Documentation: https://github.com/pzs/vuetify-toast
